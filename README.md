## Funcionalidades do software
  - Qualquer pessoa podera aunciar um produto
  - Qualquer pessoa podera fazer pedidos
#### Funcionalidades
  * Uma pessoa tem: 
    - nome
    - telefone
    - email
  * Um produto tem um:  
    - nome
    - detalhes
    - valor
    - codigo
    - disponivel
    - fotos ?
  * Umpedido tem: 
    - produto 
    - uma pedido
    - quantidade
    - local de entrega
    - observacoes
